A Raise to the Next Dimension!!

So far, we saw single dimension arrays. But it is possible to have 2D, 3D, ... ND arrays.
Relax guys, you can sit down now.
Multi dimensional arrays are useful for
  • matrix calculations
  • image representation
  • 2D game maps
A 2D array is an array of arrays! You have a number of rows, all with the same number of columns.
Initialization of a 2D array is similar to a 1D arrays. You can set the values when you declare it. 
//Type arrayName[numberOfRows][numberOfColumn];

int out_of_this_dimension[3][5];

/*

int my2d[4][3] = {{22, 23, 10},
            	  {15, 25, 13},
            	  {20, 74, 67},
            	  {11, 18, 14}};

To access elements of a two dimensional array, you do it using a pair of indices for the row and column 
cout << A[1][2];      //print an array element
A[1][2]=13;           // assign value to an array element
cin >> A[1][2];       // input element

To loop through all the elements of a 2D array you need two loops. Outer for the rows, inner for columns 
int mat[3][5], row, col ;
for (row = 0; row < 3; row++)
  for (col = 0; col < 5; col++)
    cin >> mat[row][col];

2D Arrays as Parameters

2D arrays can be passed to functions and, like regular arrays, they are passed by references so any changes to the array made during function calls are persistent.
What is different is that you can omit the number of rows BUT you MUST specifiy the number of columns 
 void print(int A[][3],int N, int M);
// ...
// Somewhere in the main

int arr[4][3];
// initialize ...

// call the print function
print(arr, 4, 3);

// Here is a working example
#include <iostream>
using namespace std; 
void print(int A[][3],int N, int M)
{
  for (R = 0; R < N; R++)
    for (C = 0; C < M; C++)
       cout << A[R][C];
}

int main ()
{
  int arr[4][3] ={{12, 29, 11},
                  {25, 25, 13},
                  {24, 64, 67},
                  {11, 18, 14}};
  print(arr,4,3);
  return 0;
}

Now that we looked at arrays, we are going to talk a little bit more about strings
So far we have used C++ strings. If you noticed, you access characters in the string using the [] operator like an array.
That is because C++ strings pretty much are arrays. They are a custom class packed with rich functions

However, before there were C++ strings, there was C++. And before there was C++ there was C. Unfortunately C did not have C++ strings (sad face). It had C-strings.

C-Strings are arrays of characters!
We signify the end of the relevant portion of the C-String using a zero-byte character ('\0')
